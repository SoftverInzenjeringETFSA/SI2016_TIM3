1. Instalirati Eclipse Neon.3 for JavaEE developers 
2. Instalirati MySql 6.3 sa svim dodacima (developer kit u instalacijskim postavkama)
3. Kroz Eclipse Marketplace instalirati jBoss tools
4. Sa oficijelne stranice hibernate.org skinuti verziju 5.2.10
5. Extraktovati skinuti zip bilo gdje i kopirati sadrzaj \hibernate-release-5.2.10.Final\lib\required foldera u 
korisnicki kreirani folder lib, unutar projekta
6. Skinuti MySql konektor, extractovati, te dodati u lib folder 
7. Napraviti refresh projekta
8. Kreirati korisnicku biblioteku i unutar nje dodati sadrzaj lib foldera
9. U path projekta dodati novokreiranu korisnicku biblioteku
10. Kreirati shemu baze podataka pod nazivom TimX (X je broj tima)
11. Dodati korisnika sa korisnickim ispravama datim u postavkama projekta na c2.etf.unsa.ba
12. Kreirati novu konekciju na bazu, ponovo prema uputama datim na c2
13. Instalirati Java Spring, kroz Eclipse Marketplace
14. Kreirati Java Spring MVC projekat
15. Dodati novi package unutar /src/main/java foldera koji ce predstavljati nas model
16. Unutar modela dodati POJO (plain old Java object) klase
17. Kreirati hibernate.cfg.xml konfiguracijski fajl, te ga popuniti odgovarajucim xml-om
18. Za kreirani model kreirati mapping fajlove, te i njih popuniti odgovarajucim xml-om
